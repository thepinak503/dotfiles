Architecture

Repository Layout

shells/Bash, Zsh, Fish entry points, core libraries, full catalogs, and modes
core/Shared detection, tool, battery, SSH, and logging helpers
apps/External application configs for Starship, Fastfetch, Git, Tmux, Lazygit, Atuin, and Brew
bin/Generators, diagnostics, benchmarks, update helpers, and docs tooling
install/Installer and uninstaller

Generated Assets

bin/generate_assets.js owns the large synchronized shell catalogs and prompt configs. Edit the generator first, then regenerate.

node bin/generate_assets.js

Modes

All modes currently resolve to ultimate.zsh. Mode selection infrastructure exists for future differentiation.
