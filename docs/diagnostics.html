Diagnostics

Centralized Error Trace — dots logcat

dots logcat is a unified error tracing tool that aggregates errors from all dotfiles components into a single, colorized, filterable stream — similar to Android's logcat.

Usage

dots logcat                  # Last 50 errors
 dots logcat -n 100           # Last 100 entries
dots logcat -f               # Real-time follow mode
dots logcat --summary        # Error count by component
dots logcat -c shell         # Shell errors only
dots logcat -S ERROR         # Only errors (no warnings/info)
dots logcat --since "2026-05-01"  # Since May 1st
dots logcat -c hyprland -S ERROR -f  # Watch Hyprland errors
dots logcat --json           # Machine-readable JSON output
dots logcat --watch          # Refresh every 5s
dots logcat --all            # Show all available entries

Follow Mode

Use dots logcat -f for real-time monitoring. This tails all log files simultaneously and color-codes each new entry by severity and component. Useful when troubleshooting during desktop use.
