Bash Architecture

The Bash configuration is built for 100% POSIX compliance while layering on modern features. It uses a modular sourcing strategy to keep startup times under 100ms.

Core Modules

  • .bashrc: The main entry point. Sets environment variables and gates mode-specific modules.
  • exports.bash: Handles PATH, GPG keys, and tool initializations (Atuin, Mise, Direnv).
  • functions.bash: A library of 160+ reusable shell utilities.
⚡ Performance: Bash uses lazy-loading for heavy aliases to ensure the prompt appears instantly.

Bootstrap Process

# Order of initialization:
1. Environment Detection (core/)
2. Global Exports (shells/bash/exports.bash)
3. Specialized Aliases (pkg_aliases.bash)
4. Starship Prompt Hook (Last to ensure PS1 control)
