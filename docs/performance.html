Startup Policy

DefaultCore aliases/functions only
Full catalogdotloadfull or DOTFILES_LOAD_FULL=1
Fastfetchenabled by default; disable with DOTFILES_FASTFETCH_ON_STARTUP=0
UpdaterDOTFILES_AUTO_UPDATE=1
StarshipCompact prompt without always-on custom command modules

Benchmark

for i in 1 2 3; do zsh -c 'zmodload zsh/datetime; s=$EPOCHREALTIME; zsh -i -c exit >/dev/null 2>&1; e=$EPOCHREALTIME; printf "zsh %.3f\n" $(( e - s ))'; done

Tuning

  • Keep DOTFILES_LOAD_FULL unset for daily shells.
  • Run dotloadfull only in sessions where the large generated catalog is useful.
  • Set DOTFILES_FASTFETCH_ON_STARTUP=0 on machines where terminal tab creation speed matters more than the startup summary.
  • Use minimal mode on remote hosts.
