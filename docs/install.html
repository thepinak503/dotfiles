Installation

Supported Systems

Primary targetArch Linux, EndeavourOS, Manjaro, Garuda, CachyOS, and related pacman systems
Secondary targetsDebian/Ubuntu, Fedora, openSUSE, Alpine, Void, generic Linux
macOSSupported through Homebrew and the macOS Starship profile

Bootstrap

curl -fsSL https://raw.githubusercontent.com/thepinak503/dotfiles/main/install.sh | sh

Manual Install

git clone https://github.com/thepinak503/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
sh install.sh
