Installation

Supported Systems

Cross-distro: works on any Linux distribution and macOS.

CategoryDistributions
Shell config (all)Arch, Debian, Ubuntu, Fedora, openSUSE, Alpine, Void, Gentoo, NixOS, Slackware, Solus, macOS, and any Linux
Desktop (Hyprland)Linux only — Arch, Fedora, openSUSE, Void, Gentoo, Debian/Ubuntu (+ manual build)
Shell-only modeServers, containers, WSL, macOS — no desktop dependencies

Install Profiles

ProfileInstalls
--shell-onlyShell config + core CLI tools (git, fzf, ripgrep, bat, eza, starship, zoxide, fastfetch, atuin)
--minimalShell + dev tools + git tools
--standard (default)Shell + dev + git + containers
--fullEverything — auto-detects Hyprland on Linux
--hyprlandHyprland/Wayland desktop packages only
--macosmacOS extras (coreutils, findutils, gnu-sed, gnu-tar, grep)

Bootstrap

curl -fsSL https://raw.githubusercontent.com/thepinak503/dotfiles/main/install.sh | sh

Shell-Only Install (servers, macOS, WSL)

curl -fsSL https://raw.githubusercontent.com/thepinak503/dotfiles/main/install.sh | sh -s -- --shell-only

Manual Install

git clone https://github.com/thepinak503/dotfiles.git ~/.local/share/dotfiles
cd ~/.local/share/dotfiles
sh install.sh
