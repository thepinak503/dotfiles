9. _x Runtime Guard System

This is the core innovation. Every alias and function uses _x to check tool availability at runtime:

# Instead of brittle:
alias dkps='docker ps'  # fails if docker missing

# We use:
alias dkps='_x docker ps'  # shows "missing: docker" if not installed

# The _x function:
_x() {
    if command -v "${1%% *}" >/dev/null 2>&1; then
        "$@"
    else
        echo "missing: $1" >&2
        return 127
    fi
}

Every command works on every system — if the tool is installed it runs, otherwise you get a clean missing: <tool> message. No more cryptic errors on cross-platform setups.
